    }
    Colores en la ramalera: {

        Fondo: #F5F5DC
    }
}
Descipcion del Recorrido: "El rondin arranca fuera del Deposito Fonobus. Luego de aqui, se dobla a la derecha para dirijirse a la calle que lleva al metrobus pero antes de entrar al metrobus, se da vuelta en U para entrar al Diagonal. Al entrar al Diagonal, se desviara a la izquierda para agarrar la Avenida Principal para hacer una casi vuelt entera en la rotonda y desviar hacia donde comienza el rondin. (Lo anterior se repetira 3 veces), al ya realizar las 3 vueltas, se pondra el cartel: A-Vuelta para luego: seguir derecho y doblar a la derecha para cruzar hacia la Avenida Principal y luego entrar a la Terminal de Retiro. Al parar alli, se termina el recorrido."

Linea 164 - Ramal B:
====================
Ramalera: {
    Opciones: {
        RONDIN: "Depo. Publico - Depo. FonoBUS - Av. Principal - T. Retiro - Depo. GTG".
        VUELTA: "Depo. Publico - Metrobus - Av. Principal - T. de Omnibus".
    }
    Colores en la ramalera: {
        Texto: #F5F5DC
        Fondo: #333333
    }
}
Descipcion del Recorrido: "El rondin arranca fuera del Deposito Publico, se desvia para pasar por la Terminal de Minibuses y llegar al Deposito Fonobus. Luego de aqui, se dobla a la derecha para dirijirse a la Terminal de Retiro y alli aguardar 2 minutos para salir nueva mente. Al salir, se desviara a la derecha para entrar en la Villa y luega pasar por el frente del Deposito Larga Distancia y mas luego pasar por el frente del Deposito de Gral. Tomas Guido y luego seguir mucho mas adelante para desviar hacia donde inicia el rondin. (Lo anterior se repetira 2 veces), al ya realizar las 2 vueltas, se pondra el cartel: B-Vuelta para luego: seguir derecho y doblar a la izquierda al cruzar con el metrobus, despues de estar en el metrobus, se sigue derecho y se estaciona en la Terminal de Omnibus ubicada en la Villa. Al parar alli, se termina el recorrido."
